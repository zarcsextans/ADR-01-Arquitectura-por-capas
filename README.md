# ADR-01-arquitectura por capas
# Mi Biblioteca

## Descripción

Biblioteca Digital es una aplicación web desarrollada para gestionar libros, usuarios, préstamos, reservas y pagos de manera eficiente. El sistema busca digitalizar los procesos de administración bibliotecaria, facilitando el acceso y control de la información.

## Tecnologías utilizadas

- ASP.NET Core MVC
- ASP.NET Core Web API
- PostgreSQL
- Arquitectura por capas
- Git y GitHub

## Funcionalidades principales

- Gestión de libros
- Administración de usuarios
- Control de préstamos y devoluciones
- Reservación de libros
- Gestión de pagos
- Consulta del catálogo digital

## Arquitectura

El proyecto utiliza una arquitectura por capas compuesta por:

- Presentación (MVC)
- API
- Servicios
- Repositorios
- Base de datos PostgreSQL

## Documentación

Las decisiones arquitectónicas del proyecto se documentan mediante ADRs (Architecture Decision Records) incluidos en este repositorio.
